The next piece is about a description list

Description List
A description list is made with the dl container. The title of the list is marked with dt while this definition is marked dd. This can be as long as you need, and will stand out from the title thanks to the structure of this list.
Special Entity Characters
These are characters such as <, ©, or &. They require the and symbol, followed by some text and then finished with a semicolon to mark what they are.
  • quot "
  • copy ©
  • amp &
  • nbsp  Something's here I promise, it's called nonbreaking space
  • rsquo ’
  • mdash —
  • #1234 Ӓ
  • lt <
  • gt >
These are all the phrases you can use to make the various entity characters.
